# Ciclos

Los ciclos son tareas repetitivas que se realizar para automatizar ciertas tareas que presentas una serie de sucesi√≥n para uno resultado.

```js
for (let step = 0; step < 5; step++) {
  // Se ejecuta 5 veces, con valores del paso 0 al 4.
  console.log('Camina un paso hacia el este');
}
```
